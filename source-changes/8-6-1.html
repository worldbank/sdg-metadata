+ <p>Work statistics for countries not using the same set of statistical standards are not comparable. As such, each series is based on a single set of standards (i.e., 13<em>th</em> or 19<em>th</em> ICLS) and contains only data comparable within and across countries, allowing data users to continue making meaningful time series analysis and international comparisons. Users should not compare data across series.</p>
